install
=====================================
### prettier 同时支持yarn 和 npm
   #### yarn 安装方式如下（感觉这点就是凑数的~）
   > yarn add prettier --dev --exact
   > # or globally
   > yarn global add prettier
   #### npm 安装方式如下
   >npm install --save-dev --save-exact prettier
   ># or globally
   >npm install --global prettier
